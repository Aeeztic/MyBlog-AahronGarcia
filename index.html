Brier History of Evolution in computing:

Analytical Engine - 1837

Designed by Charles Cabbage who's referred to as the "Father of Computers" the Analytical Engine was a Machinal computer that was supposed to be able to perform calculations. Though never completed the concept introduced the idea of a programmable machine capable of Complex Calculations

The Turing Machine - 1936

Introduced by the English Mathematician Alan Turing the Turing machine was a mathematical machine that was capable of reducing the logical structure of any computing device to its essentials.

The Complex Number Calculator (CNC) - 1940

The Complex Number Calculator designed by the Mathematician George Stibitz was considered as the forerunner of the digital computers. Presented during the American Mathematical Society conference the invention created the first example of remote access computing operation by inputting problems through a teleprinter.

Electronic Numerical Integrator and Computer(ENIAC) - 1943-1946

Constructed by by J. Presper Eckert and John Mauchly the ENIAC was accepted by many as the first ever Completed and Functional digital computer. It was a powerful calculating device generally used for computing values for artillery range tables during WW2.

The First Super Computer - 1964

The first ever super computer developed by Control Data Corp was a machine that was designed to handle complex scientific calculations, with the speed of 3million floating-point operations per second it was considered as the fastest of the time.

First Windows Operating System - 1985

During 1985 Microsoft introduced the first ever version of Windows 1.0. What made this release notable is because windows heavily relied on the use of mouses where in that time people only saw the mouse as an input device.

World Wide Web - 1989

Created by Tim Berners-Lee the world wide web started as a normal information sharing platform through researchers and scientists but its worth was recognized and is now shaped into the web we use now.

Introduction of IT Basics

Information Technology is a wide field, it refers to different areas of study within the field of technology. Used by the different industries IT is utilized to access information and business operations. Companies nowadays uses IT to manage and process data as its efficient and Useful.

Hardware components

Central Processing Unit (CPU)

Referred to as the brain of a computer. The CPU or also called as the central processor handles the execution of the programs and interprets the data that comes the input devices and sends to the output devices.

Random Access Memory (RAM)

The ram is also an important part of computer as it is typically used to store data. It allows the CPU to access data faster. The more ram your computer has the more task its capable of handling without slowing it down.

Storage Devices (HDD/SDD)

The storage devices is where data is permanently stored. There are 2 storage devices for your computer which are The Hard Disk Drive and the Solid State Drive. Both storage devices differs from each other, HDD uses a spinning disk and the SDD uses flash memory.

Motherboard

The motherboard contains the main circuit of your computer that's why its often referred as the Heart of a computer. It is the main connection that allow other part of the computer to communicate.

Power Supply Unit (PSU)

The power supply unit power storage of a computer. It provides the computer the necessary about of electricity to function the device. Its also important as helps keep the electrical flow of the computer and help keep it stable.

Graphic Processing Unit(GPU)

The GPU ensures the rendering of images, videos, and animations of your computer. The GPU handles a heavy task as its responsible for processing visuals for your computer. The GPU is also important for heavy task such as gaming, video, editing, and other graphic-intensive applications.

Software

Software is a computer program that provides a set of instructions to execute a user’s commands and tell the computer what to do. Software is also a collection of data that are used to run machines and perform execution of systems for the end-user.

Types of Software's

  1. System Software - refers to operating systems like Windows, IOS, macOS where it directly operates the computer hardware and ensures the functionality to the users as well as to the other software.
  2. Application software - refers to software that performs its functionality according to its purpose. Unlike System software where it directly involves into hardware to provide functionality the Application software operates its designed task to the user.

Networking

Networking refers to process of connecting devices such as a computer to share sources and information. Networking is an efficient way of exchanging data across different platforms. It is widefield used by Enterprises that consist of many forms such as building and maintaining networks.

Key Components

  • Network Devices - Hardware components that allow connection between a computer and the internet network.
  • Links - It can be defined as a chain that allows information travel between devices
  • Communication Protocols - Are set of rules that devices follow when sharing information
  • Network Defense - Are measures taken to secure the content and privacy of the data and information
